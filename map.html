Lembrando que o map não modifica o array original ele cria um novo array

Sintaxe

array.map(callback, thisArg)

Vamos pegar uma array e receber 2 itens o callback, função a ser executada em casa elemento

callback(item, index, array)

Trabalhando com forEach

O forEach tem a mesma sintaxe do map porem eu preciso por a operacao do forEach dentro de alguma constante pois ele nao retorna nada, entao preciso armazenar esse resultado em algum lugar

Trabalhando com filter

Voce coloca todo seu array dentro de um filtro,e apenas os itens responderem a determinada condicao sao retornados

Trabalhando com reduce
