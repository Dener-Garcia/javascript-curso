Injetando elementos da DOM

Outra forma de injetar elementos sem utilizar o innerHTML e o outerHTML é criar elementos dividindo-os em seus componentes fundamentais

  1. Crie o elemento com .createElement()
  2. Crie o nó de texto que vai carregar o conteudo dentro do elemento .createTextNode()
  3. Colocar o nó de texto dentro do elemento e no mesmo lugar correto do documento
  4. Adicione o elemento a árvore da DOM
