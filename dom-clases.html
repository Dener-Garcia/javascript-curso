Repare que o classList é um metodo de somente leitura, então para conseguir altera-lo existe alguns metodos para isso

Métodos para o classList

  • add - Adiciona classes
  • remove - Remova classes
  • toggle - Alterna entre adicionar e remover uma classe
  • contains - Verifica se existe a classe para aquele elemento
