Estrutura de uma função com return

O return serve para parar a função e vai retornar o valor após ele, isso significa que aquela função ela vai rodar e ela vai ter um valor armezanado que você pode utilizar em outras funções por exemplo

Ex: function nome(parametros){
//instrunções
return }

Exfunction digaOla(){
return "ola"
}

const x = digaOla()
console.log(x)

Basicamente é uma função passada como argumento para outra, utilizando callback você tem mais controle da ordem dde chamadas

const calcula = function(operacao, num1, num2){
return operacao(num1,num2)
}

const soma = function(num1, num2){
return num1 + num2
}

const sub = function(num1, num2){
return num1 + num2
}

const resultSoma = calc(soma, 1, 2)

Estrutura de uma função com this

Quando temos um objeto que tem uma função dentro dele nós chamamos essa função de metódo

Veja mais no arquivo js dessa aula funcao.js

O this retorna o firstName do objeto pessoa

const pessoa = {
firstName: "Andre",
lastName: "Soares",
id: "1",
fullName: function(){ return this.firstName + "" + this.lastName },
gedId: function(){ return this.id; }
}
pessoa.fullName() // "Andre Soares"
pessoa.gedId() // 1
