Trabalhando com loop FOR

Um loop é usado para percorrer arrays e fazer a ação a 1, muitos ou todos objetos. Um laço também serve para rodar um código enquanto a condição que criamos for verdadeira

Laço FOR quando já sabemos a quantidade de vezes que o código vai rodar usamos esse laço

Ex: for (var i = 0; i < 10; i++) { console.log(i); }

Nesse código enquanto i for menor que 10 ele se repetira até i for = 10, repare que toda fez que o código se repete poderemos ver o resultado já que temos um console log dentro desse laço

Crie uma funcao que cria um range de 5 numeros apartir do numero digitado, ex digitei 5 vai gerar uma array de 5,6,7,8,9,10

Laço WHILE quando queremos que o loop rode ate que alguma condição mude, geralmente a variável inicializadora esta antes do próprio loop

Ex: var i = 0
while (i < 10) { console.log(i);
i++ }

No código acima teremos o mesmo principio, porém se tivermos uma hipotése de que i fosse = 10, o código nunca rodaria certo, para corrigir isso podemos usar o do while

Ex: var i = 10
do { i ++ console.log(i)}
while (i > 10) }
