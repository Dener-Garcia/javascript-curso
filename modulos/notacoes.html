Podemos usar a notação Dot Nation ex: curso.nomeDoCurso, dessa forma vamos acessar o nome do curso do objeto curso

Algumas vezes nosso objeto pode vir de fora do nosso javascript por exemplo um PHP, e nele pode ter um objeto ou variavel escrita com uma sintaxe estranha para o JavaScript ex: curso.WP:nomeDoCurso, nesse tipo de notação quando o JavaScript encontrar os dois ponto : ele provávelmente vai dar erro

Para resolver isso usamos a Bracket notation onde vamos envolver nosso objeto ou váriavel por aspas ex: ["curso.WP:nomeDoCurso"]dessa forma vamos acessar o nome do curso do objeto curso sem erros convertendo a propriedade para uma string
