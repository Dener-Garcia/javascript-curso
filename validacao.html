Validação e depuração de javascript

Voltar

Podemos usar varias ferramentas como o console.log, console.error e ate ferramentas mais avançadas do proprio navegador

  • No navegador na aba SOURCE temos um ótimo exemplo para depuração pois ele mostra exatamente como o navegador vai executar o código linha a linha
  • JSlint é uma ferramenta online ele identifica caractere por caractere e te diz tudo que esta fazendo errado na linguagem mesmo que seu script ainda funciona
  • ESLint é uma ferramenta que trabalha por linha de comando no terminal ou mesmo dentro do VSCODE através de extensão

Minifique seu código após tudo pronto

Você pode deixar seu arquivo menor para o navegador, isso melhora a performance do seu site ou app, use o Minifier.org

Outra forma de minificar é usando extensão do VSCODE ou por linha de comando com o minifier pelo npm
